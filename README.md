# go-summarizer

Summarize web page by ChatGPT

## Quick start

```shell
make run URL="http://example.com"
```

## Usage

```shell
go run . --url "http://example.com" --lang "English"
```

## Install

```shell
# Install as `go-summarizer`
make install
go-summarizer --url "http://example.com" --lang "English"
```
